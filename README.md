# Idea general

El usuario inicia sesión y puede registrar objetos que desea recordar dónde dejó.

# Cada registro contiene:

📷 Fotografía del objeto (desde cámara o galería)
🏷 Nombre del objeto
📂 Categoría
📍 Lugar donde lo dejó (descripción)
🌎 Originalmente se implementó el guardado y control de coordenadas GPS pero debido
    a limitaciones con el conversor de web a app esta funcionalidad fue eliminada
📅 Fecha y hora del registro

Posteriormente puede buscar un objeto y visualizar toda la información.
